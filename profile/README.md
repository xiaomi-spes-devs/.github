## Xiaomi-Spes-Devs

This organization contains repositories to build AOSP ROMs for Redmi Note 11/NFC (spes/spesn). Currently supporting: Android 16 QPR2

* [**Device Tree (spes)**](https://github.com/xiaomi-spes-devs/android_device_xiaomi_spes.git) (`android_device_xiaomi_spes`)
* [**Vendor Tree (spes)**](https://github.com/xiaomi-spes-devs//android_vendor_xiaomi_spes.git) (`android_vendor_xiaomi_spes`)
* [**Kernel Sources**](https://github.com/xiaomi-spes-devs/android_kernel_xiaomi_sm6225.git) (`android_kernel_xiaomi_sm6225`)

### Other required repositories
* [**Xiaomi Hardware**](https://github.com/lineageos/android_hardware_xiaomi.git) (`android_hardware_xiaomi`)
